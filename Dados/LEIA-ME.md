# 📁 Dados — Titãs Sinergy

Pasta central de atualização do sistema.
Para atualizar qualquer aba, edite o arquivo correspondente e peça ao Claude.

---

## 📊 Aba 360° → pasta `360/`

**Arquivo:** `360/360_atual.json`

**O que atualizar todo mês:**
| Campo | O que é | Exemplo |
|---|---|---|
| `periodo` | Mês/ano por extenso | "Mai 2026" |
| `diasDecorridos` | Quantos dias úteis já passaram | 15 |
| `diasMes` | Total de dias do mês | 31 |
| `meta_rede` | Meta total da rede no mês | 3000000 |
| `total` (por loja) | Faturamento acumulado | 180000 |
| `ped` (por loja) | Pedidos acumulados | 145 |
| `cancel` (por loja) | Cancelamentos | 3 |
| `agend` (por loja) | Taxa agendamento (decimal) | 0.38 = 38% |
| `agendCount` (por loja) | Qtd absoluta agendamentos | 55 |
| `top` (por loja) | Top 3 vendedores | nome, iniciais, total, pedidos |

**Como pedir atualização ao Claude:**
> "atualize o 360 com os dados da pasta"
